🐍 Chapter 02. 파이썬 기초

파이썬 프로그래밍의 핵심 기초 문법을 학습합니다. 조건문, 반복문부터 함수, 클래스까지 다양한 예제를 통해 확인해 보세요.

1. 조건문 (If)

조건에 따라 다른 코드를 실행합니다. (`if`, `elif`, `else`)

score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
else:
    print("C")
# 출력: B

                                        

                                    

                                    

                                        
2. 반복문 (Loops)

                                        
`while`은 조건이 참인 동안, `for`는 범위를 순회하며 반복합니다.

                                        
While 예제 - 구구단 7단

                                        

                                            

                                                
                                                
                                            

                                            
a = 7
i = 1
while i <= 9:
    print(a, 'X', i, '=', i*a )
    i = i + 1 # i를 1 증가시켜 무한 루프 방지

                                            

                                        

                                        
For 예제 - 구구단 7단

                                        

                                            

                                                
                                                
                                            

                                            
a = 7
for i in range(1, 10): # 1부터 9까지
    print(a, 'X', i, '=', i*a)

                                            

                                        

                                    

                                


                                
                                

                                    

                                        
3. 자료형 (Data Types)

                                        
데이터를 담는 여러가지 구조입니다.

                                        
                                        
리스트 (List)

                                        
순서가 있고, 수정 가능한 데이터의 모음입니다. `[]`를 사용합니다.

                                        

                                            

                                                
                                                
                                            

                                            
hobby = ['독서', '등산', '바둑']
hobby.append('수영')         # 맨 뒤에 '수영' 추가
hobby.insert(1, '여행')      # 1번 인덱스에 '여행' 삽입
print(hobby)                 # ['독서', '여행', '등산', '바둑', '수영']
print(hobby.index('등산'))   # '등산'의 위치(인덱스) 반환: 2

                                            

                                        


                                        
튜플 (Tuple)

                                        
순서가 있지만, 수정이 불가능한 데이터의 모음입니다. `()`를 사용합니다.


                                        
딕셔너리 (Dictionary)

                                        
`Key:Value` 쌍으로 이루어진 데이터의 모음입니다. `{}`를 사용합니다.

                                        

                                            

                                                
                                                
                                            

                                            
# {Key:Value}
dictContainer = {'이름':'홍길동', '나이':25, '주소':'연희로'}

# 값 변경
dictContainer['나이'] = 26

# 새 값 추가
dictContainer['혈액형'] = 'AB'

# 값 삭제
del dictContainer['주소']

print(dictContainer) 
# 출력: {'이름': '홍길동', '나이': 26, '혈액형': 'AB'}

                                            

                                        

                                        
                                        
인덱싱 및 슬라이싱

                                        
데이터의 특정 위치(인덱스)나 범위(슬라이싱)를 선택합니다.

                                        

                                            

                                                
                                                
                                            

                                            
alpha = ['a', 'b', 'c', 'd', 'e', 'f', 'g', 'h']

print(f"alpha[2:5] -> {alpha[2:5]}")   # 2번 인덱스부터 5번 *이전*까지
print(f"alpha[3:]  -> {alpha[3:]}")    # 3번 인덱스부터 끝까지
print(f"alpha[:5]  -> {alpha[:5]}")    # 처음부터 5번 *이전*까지
print(f"alpha[-2:] -> {alpha[-2:]}")   # 뒤에서 2번째부터 끝까지
print(f"alpha[-1]  -> {alpha[-1]}")    # 맨 마지막 요소

                                            

                                        

                                    

                                

                                
                                

                                    

                                        
4. 함수 (Function)

                                        
특정 작업을 수행하는 코드 블록입니다. `def`로 정의합니다.

                                        

                                            

                                                
                                                
                                            

                                            
# 매개변수에 기본값(default) 지정
def setSalary(name, pay = 200):
    print(name, '직원 급여 : ', pay, '만원')

setSalary('박찬호', 400) # 박찬호 직원 급여 :  400 만원
setSalary('박지성')       # 박지성 직원 급여 :  200 만원

# 키워드 인수를 사용하면 순서가 바뀌어도 됨
def printMemberInfo(name, email, major, grade):
    print('name :', name, ', email :', email)

printMemberInfo(grade=4, major='computer', email='abc@gmail.com', name='Mr. hong')
# 출력: name : Mr. hong , email : abc@gmail.com

                                            

                                        

                                    

                                    

                                        
5. 람다(Lambda), Map, Filter

                                        
    
                                            
  • **람다:** 한 줄로 된 간단한 익명 함수. `lambda 인수 : 표현식`
    • 
                                            
  • **Map:** 리스트의 각 요소에 함수를 적용. `map(함수, 리스트)`
    • 
                                            
  • **Filter:** 리스트의 각 요소 중 함수 결과가 `True`인 것만 반환. `filter(함수, 리스트)`
    • 
                                        

                                        

                                            

                                                
                                                
                                            

                                            
b = [1, 4, 7, 9, 12]

# 람다 함수 예제
add_two = lambda x : x + 2
print(f"람다 결과: {add_two(4)}")

# Map 예제
result_map = list( map( lambda x : x + 2, b ) )
print(f"Map 결과: {result_map}")

# Filter 예제 (짝수만)
result_filter = list( filter( lambda x : x % 2 == 0, b ) )
print(f"Filter 결과: {result_filter}")

                                            

                                        

                                    

                                    

                                        
6. 리스트 컴프리헨션 (List Comprehension)

                                        
리스트를 간결하게 생성하는 방법입니다.

                                        

                                            

                                                
                                                
                                            

                                            
b = [1, 4, 7, 9, 12]

# b의 각 요소에 2를 더한 새 리스트 생성
result = [ x + 2 for x in b ]
print(f"기본: {result}")

# b의 요소 중 홀수(x % 2 == 1)에만 2를 더한 새 리스트 생성
result_odd = [ x + 2 for x in b if x % 2 == 1 ]
print(f"If 조건: {result_odd}")

                                            

                                        

                                    

                                

                                
                                

                                    

                                        
7. 클래스(Class)와 상속(Inheritance)

                                        
    
                                            
  • **클래스:** 객체를 만들기 위한 설계도.
    • 
                                            
  • **객체:** 클래스로부터 생성된 실체.
    • 
                                            
  • **상속:** 부모 클래스의 속성과 기능을 자식 클래스가 물려받는 것.
    • 
                                        

                                        
예제 - 클래스

                                        

                                            

                                                
                                                
                                            

                                            
class Calculator:
    # 생성자 (객체가 생성될 때 실행됨)
    def __init__(self, n1, n2):
        self.num1 = n1
        self.num2 = n2
    
    # 메서드 (객체의 기능)
    def add(self):
        print(f"더하기: {self.num1 + self.num2}")

# 객체 생성
calc1 = Calculator(10, 20)
calc1.add() # 출력: 30

                                            

                                        

                                        
예제 - 상속

                                        

                                            

                                                
                                                
                                            

                                            
# 부모 클래스
class Rabbit :   
    def __init__(self, a):
        self.shape  = a
    def goto(self, x, y) :
        print(self.shape, 'Rabbit’s Going to', x, y)

# 자식 클래스 (Rabbit을 상속)
class HouseRabbit(Rabbit) :
    def __init__(self, a, b):
        self.shape  = a
        self.owner  = b
    # 자식만의 고유 메서드
    def feed(self) :
        print(self.owner, "are feeding the rabbit")

hRabbit = HouseRabbit("House", "Alice")
hRabbit.goto(100,100) # 부모의 goto 메서드 사용
hRabbit.feed()        # 자신의 feed 메서드 사용

3. 데이터 선택 (Slicing)

                                        
                                        
.loc (Label-based)

                                        
이름(레이블) 기반으로 데이터를 선택합니다.

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
data = {'이름': ['한빛', '한결', '한라', '준호', '가원', '규민'],
        '성별': ['남자', '남자', '여자', '남자', '여자', '남자'],
        '나이': [20, 21, 20, 30, 24, 23],
        '키': [180, 177, 160, 183, 162, 179]}
df = pd.DataFrame(data)

print("--- 2번, 3번, 4번 행 ---")
print(df.loc[2:4]) # loc는 끝점(4) 포함

print("\n--- '키', '나이' 컬럼 ---")
print(df.loc[:, ['키', '나이']])

print("\n--- '키' > 170 인 데이터 ---")
print(df.loc[df['키'] > 170])

print("\n--- '성별' == '여자' & '키' > 160 ---")
print(df.loc[(df['성별'] == '여자') & (df['키'] > 160)])

4. 데이터 조작

                                        
    
                                            
  • .drop([인덱스/컬럼], axis=0/1): 행(0) 또는 열(1) 삭제
    • 
                                            
  • .sort_values(by='컬럼명'): 특정 컬럼 기준으로 정렬
    • 
                                            
  • .set_index('컬럼명'): 특정 컬럼을 인덱스로 설정
    • 
                                            
  • .reset_index(): 인덱스를 리셋하고 기존 인덱스를 컬럼으로 추가
    • 
                                            
  • .rename(columns={'A':'B'}): 컬럼명 변경
    • 
                                        

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
data = {'이름': ['한빛', '한결', '한라', '준호'],
        '성별': ['남자', '남자', '여자', '남자'],
        '나이': [20, 21, 20, 30]}
df = pd.DataFrame(data)
print("--- 원본 ---")
print(df)

print("\n--- .drop(1, axis=0) : 1번 행 삭제 ---")
df_dropped_row = df.drop(1, axis=0)
print(df_dropped_row)

print("\n--- .drop('성별', axis=1) : '성별' 열 삭제 ---")
df_dropped_col = df.drop('성별', axis=1)
print(df_dropped_col)

print("\n--- .sort_values(by='나이', ascending=False) : 나이 내림차순 정렬 ---")
df_sorted = df.sort_values(by='나이', ascending=False)
print(df_sorted)

print("\n--- .rename(columns={'나이':'Age'}) : '나이' -> 'Age' ---")
df_renamed = df.rename(columns={'나이':'Age', '이름':'Name'})
print(df_renamed)

print("\n--- .set_index('이름') : '이름'을 인덱스로 ---")
df_indexed = df.set_index('이름')
print(df_indexed)

print("\n--- .reset_index() : 인덱스 리셋 ---")
df_reset = df_indexed.reset_index()
print(df_reset)

                                            

                                        

                                    

                                    

                                        
5. 그룹화 (Groupby)

                                        
데이터를 특정 기준으로 묶어 통계치를 계산합니다.

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
data = {'이름': ['한빛', '한결', '한라', '준호', '가원', '규민'],
        '성별': ['남자', '남자', '여자', '남자', '여자', '남자'],
        '나이': [20, 21, 20, 30, 24, 23],
        '키': [180, 177, 160, 183, 162, 179]}
df = pd.DataFrame(data)

print("--- 성별 키 평균 ---")
mean_by_gender = df.groupby(['성별'])['키'].mean()
print(mean_by_gender)

print("\n--- 성별, 나이별 키 평균 ---")
mean_by_group = df.groupby(['성별','나이'])['키'].mean()
print(mean_by_group)

                                            

                                        

                                    

                                

                                
                                

                                    

                                        
6. 함수 적용 (Apply & Map)

                                        
    
                                            
  • .apply(함수): 행 또는 열 전체에 복잡한 함수를 적용
    • 
                                            
  • .map(딕셔너리): 시리즈의 각 값에 딕셔너리를 매핑하여 값 변환
    • 
                                        

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
df_apply = pd.DataFrame({ "a": [10, 20, 30], "b": [20, 30, 40] })

def my_sq(x):
    return x ** 2

print("--- Apply 예제 ---")
sq = df_apply['a'].apply(my_sq)
print(sq)

df_map = pd.DataFrame({'성별': ['남자', '여자', '남자']})
rep_cond = {'남자':1, '여자':0}

print("\n--- Map 예제 ---")
df_map['성별_숫자'] = df_map['성별'].map(rep_cond)
print(df_map)

                                            

                                        

                                    

                                

                                
                                

                                    

                                        
7. 데이터 병합 (Concat & Merge)

                                        
Concat (연결)

                                        
데이터프레임을 단순히 위아래(axis=0) 또는 좌우(axis=1)로 이어 붙입니다.

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
df1 = pd.DataFrame({'A': ['A0', 'A1'], 'B': ['B0', 'B1']})
df2 = pd.DataFrame({'A': ['A2', 'A3'], 'B': ['B2', 'B3']})

print("--- Concat (행) ---")
row_concat = pd.concat([df1, df2], ignore_index=True)
print(row_concat)

                                            

                                        

                                        
                                        
Merge (병합)

                                        
특정 공통 키(Key)를 기준으로 두 데이터프레임을 조합합니다.

                                        

                                            

                                                
                                                
                                            

                                            
import pandas as pd
person = pd.DataFrame({'ident': ['DR-1', 'DR-3', 'MSK-4'],
                     'personal': ['Dyer', 'Lake', 'Pabodie']})
visited = pd.DataFrame({'ident': ['DR-1', 'DR-3', 'MSK-4'],
                      'site': ['Station A', 'Station B', 'Station C']})

print("\n--- Merge 예제 ---")
merged_df = person.merge(visited, left_on='ident', right_on='ident')
print(merged_df)
